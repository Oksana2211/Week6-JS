Ответы на вопросы

  1. Что называется массивом?

    2. Ответ:
    Массив – это упорядоченная коллекция значений. Значения в массиве называются элементами, и каждый элемент характеризуется числовой позицией (или строкой если речь идет о ассоциативных массивах) в массиве, которая называется индексом (ключ для ассоциативных).

  2. С какого порядкового номера начинаются массивы?

    3. Ответ:
    Элементы массива нумеруются слева-направо, начиная с 0.

  3. Сколько значений может иметь массив?

    4. Ответ:
    Элементов может быть сколько угодно, они могут быть какими угодно.

  4. Как можно получить значение текстового поля из данной формы? (Форма одна)

    let inputValue = inputText.value;

  8. Может ли массив состоять из элементов разных типов?

    9. Ответ:
    Массивы в языке JavaScript являются нетипизированными: элементы массива могут иметь любой тип, причем разные элементы одного и того же массива могут иметь разные типы.
